# Snippity

Sublime Text 2 snippets for things I end up typing a lot

## RSpec Snippets

- it - it block
- its - it block stub
- describe - describe block
- let - let block
- factory - factory girl create
- trait - trait block
- pry - binding.pry

## Jasmine Snippets

- describe - describe block
- it - it block

## HTML Snippets

- divc - div with class

## ERB Snippets

- # comment
- = expression
- - logic
- sff - simple form for block
- sfi - simple form input
